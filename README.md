## Installazione dei requisiti
E' innanzitutto necessario scaricare tutti i requisiti per l'esecuzione del progetto. 
Da linea di comando, spostarsi nella directory principale del progetto ed esefuireil comando:

```pip install -r requirements.txt```

Se il comando precendente in Windows resituisce un errore provare con:

```py -m pip install -r requirements.txt```

E' inoltre necessaria la versione 8.4.2 di Swi-Prolog

Link pagina dei download di Swi-Prolog: ```https://www.swi-prolog.org/download/stable/bin/swipl-8.4.2-1.x64.exe.envelope```

## Utilizzo del Sistema 
Per avviare il sistema, spostarsi tramite linea di comando nella directory  ```src/``` 
ed eseguire il comando: 

 ```python main.py```
